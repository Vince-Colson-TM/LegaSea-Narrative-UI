Narrative UI for AI Fossil Identification

Vince Colson
Fontys ICT
LegaSea Project
2025
Project Plan

Abstract

This research investigates narrative-based user interfaces as an alternative to traditional form-based data collection methods for AI fossil identification. As part of the LegaSea project—a collaborative initiative to reconstruct Dutch Ice Age biomes through AI-assisted citizen science—this study addresses limitations in current fossil data collection platforms. The oervondstchecker.nl platform, while successful in accumulating over 75,000 citizen science images, faces challenges including static workflows, form-based constraints, and limited user engagement. This research explores whether conversational, chat-like interfaces can improve user engagement, data completeness, and trust in AI systems through real-time feedback, adaptive questioning, and transparent AI communication.

Using a mixed-methods approach, three functional prototypes will be developed and tested: a traditional form-based interface (baseline), a basic conversational interface, and a narrative-enhanced interface with storytelling elements. User testing with fossil enthusiasts and citizen scientists will evaluate engagement, data quality, and trust metrics to determine the effectiveness of narrative UI approaches in scientific data collection contexts.

Current Project Status

Phase 1: Project Planning & Literature Review (Weeks 1-10)

This research is currently in the initial planning and background research phase. The comprehensive project plan has been developed, outlining the three-prototype approach and mixed-methods evaluation strategy. Literature review is ongoing to establish theoretical foundations for narrative UI design in citizen science contexts.

Next Steps: Prototype development will begin in Week 10, followed by user testing preparation and execution in Weeks 18-24.
